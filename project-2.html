Project Overview

I developed an autonomous robot using GPS, ultrasonic sensors, and Arduino to detect and map potholes with 95% accuracy. The robot achieved precise location tracking within a 3-meter radius and measured pothole depth with a 2 cm margin of error, classifying severity into three levels. It navigates autonomously, avoiding obstacles with 90% success, and wirelessly transmits data to a central web app within a 30-meter range using ESP32. The robot logs pothole data with 100% uptime and features a durable IP65-rated enclosure with an 8-hour battery life for field use.
